EnemyUIName¶

In Yuri’s Revenge, allies and enemies alike can see the name of any unit and structure. In the earlier games, the tooltip would for example show “Enemy Structure” instead. This setting is useful to hide the name of fake structures.

[BuildingType]►EnemyUIName= (csf string)

Optional name used for enemy buildings. Observers, allies, and players who infiltrated the building will see the real name defined by UIName, all other players will see this name.

Note

This tag is currently limited to BuildingTypes. It might be expanded to other TechnoTypes in the future.

New in version 0.B.
