Unit or Structure Lost EVA Report¶

If a unit or a structure is destroyed, EVA announces to the owning player that they have lost it. This EVA report can now be customized or turned off.

Note

The EVA message is not played for units or structures if DontScore=yes, Insignificant=yes or Spawned=yes.

[TechnoType]►EVA.Lost= (evamd entry)
The message played when a unit or strucure of this type is destroyed. Use none to disable the report. Defaults to none for structures, to EVA_UnitLost otherwise.

New in version 0.5.

Changed in version 3.0.
