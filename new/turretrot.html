Turret ROT¶

Ares supports to define a value separate from ROT to set the rotation speed for a unit’s turret.

[TechnoType]►TurretROT= (integer)
Defines the rate of turn for the turret. Valid values range from 0 to 127. Defaults to ROT.

New in version 0.D.
