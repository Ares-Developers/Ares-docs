Unit Lost EVA Report¶

If a unit is destroyed, EVA announces that the player has lost a unit. This EVA report can now be customized or turned off.

Note

The EVA message is not played for units if DontScore=yes, Insignificant=yes or Spawned=yes.

[Unit]►EVA.Lost= (evamd entry)
The message played when a unit of this type is destroyed. Use none to disable the report. Defaults to EVA_UnitLost.

New in version 0.5.
