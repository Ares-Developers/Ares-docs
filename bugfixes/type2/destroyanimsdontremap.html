DestroyAnims Don’t Remap¶

The remappable parts of animations played via the use of the DestroyAnims flag (when a building is destroyed) used to remain red in color they would not get remapped to the owning player’s house color. Now they will.

Note

The game files already include suitable animations for some buildings, however some of these (the Allied Barracks and Prism Tower) use the red palette indexes rather than the remappable indexes so you will need to recolor the images.

New in version 0.1.
